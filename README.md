# abc-extension

Chrome 扩展程序 - 书签管理、页面能力提取、AI 总结

## 功能

- 📚 **书签管理**：保存当前页面到书签，查看历史书签
- 🔍 **页面能力提取**：提取页面元信息、链接、图片、脚本等
- 🤖 **AI 总结**：使用 OpenAI API 对页面内容进行智能总结

## 安装

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本文件夹

## 配置

在插件弹出窗口的「AI 总结」标签页输入你的 OpenAI API Key

## 技术栈

- Manifest V3
- Chrome Bookmarks API
- Chrome Scripting API
- OpenAI API
